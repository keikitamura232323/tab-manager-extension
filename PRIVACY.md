# TabFlow プライバシーポリシー

## データ収集について

TabFlow は以下のデータのみを処理します。

### 処理するデータ（すべてローカル保存）

- **タブ情報**: タブのタイトル、URL、グループ情報を表示のために参照します。外部送信は一切行いません。
- **ユーザー設定**: 表示モード、ダークモード、コンパクトモード等の設定を Chrome のローカルストレージに保存します。
- **パネル位置・サイズ**: Floating Panel の位置とサイズをローカルストレージに保存します。

### 収集しないデータ

- ユーザー名、メールアドレス、パスワード
- クレジットカード情報
- 閲覧履歴の集約・分析
- 位置情報、生体認証データ
- その他の個人を特定できる情報

### 外部通信

- TabFlow はユーザーデータを外部サーバーに送信しません。
- アナリティクス、トラッキングは一切行いません。
- すべてのデータ処理はブラウザ内で完結します。

### データの削除

- 拡張機能をアンインストールすると、保存データは自動的に削除されます。
- 設定画面からいつでもリセット可能です。

### セキュリティ

- Manifest V3 準拠
- Content Security Policy（CSP）によるスクリプト実行制限
- インラインスクリプト禁止（XSS 対策）
- iframe sandbox によるセキュリティ隔離
